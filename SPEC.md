# Weather App — Spec

## Goal

A browser-based weather app that shows current conditions and a 5-day forecast
for any searched city. Built step-by-step as a learning project.

## Steps

1. **Starting** — Project files, React + Vite setup, git + GitHub
2. **Layout** — Build the interface with React components
3. **Fetching Data** — Connect to a weather API (OpenWeatherMap)
4. **Forecast** — Add the 5-day forecast panel
5. **Polish** — Refine UI using a DESIGN.md design system
6. **Deploying** — Deploy to the web (Vercel)
7. **Next** — Ideas for going further

## Non-goals

- No backend / server — all API calls from the browser
- No user accounts or saved history (for now)
- No native mobile app
